# Forkify - Recipe Website

<img width="1440" alt="Screen Shot 2019-11-25 at 5 22 44 PM" src="https://user-images.githubusercontent.com/56647167/69544049-5c8eb480-0fa8-11ea-8741-e528f81c9b1e.png">
This is a simple recipe render website that displays the follwing

* Recipes
* Ingredient for each recipe
* Shopping Cart
* Favorite Recipes

<img width="1440" alt="Screen Shot 2019-11-25 at 5 23 33 PM" src="https://user-images.githubusercontent.com/56647167/69544036-526cb600-0fa8-11ea-97b9-a085e3c76de3.png">

---

 ### Technology used :
This web app was built using:
*  Javascript (Vanilla)
* Node Package Manager (NPM)
* Babel (To convert ES6 to ES5)
* CSS3 
* HTML
* Webpack - To compile all the pakcages and js files to build a adequate bundle (bundle.js) to offer a much better and seemless execution of code
